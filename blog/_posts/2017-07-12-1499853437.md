```sh
$ docker network rm \
  $(docker network ls \
    | sed 1d \
    | cut --delimiter=' ' --fields=1)
```

あんまりよくわかっていない。  
削除できないものは
```
Error response from daemon: bridge is a pre-defined network and cannot be removed
```
とかでエラーになるので多分大丈夫なはず。
